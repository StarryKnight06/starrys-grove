HTML & CSS Guides

  • w2schools: a great place to start if you're new to coding! teaches HTML, CSS, JavaScript, & more

  • Absolute beginner's guide to Neocities:
    a specific tutorial on how to get started on neocities

  • Learn HTML & CSS:
    a collection of HTML & CSS guides, pretty easy to understand

  • 32-Bit Cafe: an intro to making your own website along with some additional resources

  • Web Design in 4 minutes:
    a hands on guide to web design, also gives you more resources on HTML and CSS

  • HTML Semantic Elements: semantic elements are elements with meaning like "aside" or "article"

  • CSS Terms and Definitions: helpful reference article

Graphics

  • pixel-soup: lots of cute gifs, pngs, backgrounds, and dividers

  • tora*7kg: a variety of graphics made by the artist!

  • Betty's Graphics: HUGE archive of graphics

  • pixel safari: dividers, favicons, blinkies, buttons, etc.

  • GifCities: archive of old web GIFs

  • Bonnibel's Graphic Collection: I used this a lot to find GIFs n such

  • Glitter-Graphics: more GIFs, dividers, and icons

  • TopeconHeroes: icons, pngs, illustrations (use the buttons at the top to navigate)

  • animated glitter graphics: found a lot of my dividers and lil icons here

  • Tiny Backgrounds: a collection of repeating / tiled backgrounds

  • King-Lulu-Deer: artist that makes pixel art (please credit them if you use their work!)

  • sadgrl.online: more tiled backgrounds I've used (this site also has a lot of HTML resources and more)

  • Cake Batter: collection of graphics. has a lot of everything

  • HomuHoard: large graphic collection

Things to Add to Your Site

  • Simple Javascript/CSS Jukebox Tutorial: this is the code I used for the music player, made by Solaria

  • GifyPet: make a pet for your webpage like my pet flower, Sunny

  • Atabook: a free, customizable guestbook (I use this, it's cool :3)

  • imood updater: status / mood updater

  • FC2 Counter: free visitor counter and online viewer counter

  • Shroom.Ink: adopt a mushroom to protect your website

Misc. Tips & Help

  • Getting started with Visual Studio Code: not a HTML guide BUT it's a good code editor and it's free. highly recommend using VSC

  • File Naming Tips: guide on how you can consistanly name your files (do this early on so you don't have to rename all your files later)

  • 32-Bit Cafe Forum: a great community for web hobbyists and professionals (has a lot of resources, you can also ask questions and get help on things here)

  • Common HTML Errors & How to Fix / Avoid Them: the name says all, a very helpful resource

  • Neocities Tips: guide on how to use external widgets, like status cafe, on your website with a free account and how to automatically upload files to your neocities via GitHub. really helpful guide. I use both methods on my website

Helpful Tools

  • catbox: you can upload up to 200MB of files here and link them elsewhere

  • HTML Cheat Sheet: quick references for HTML, CSS, and JavaScript (I use this ALOT)

  • Kaomoji: a collection of Japanese emoticons like this one (*・ω・)ﾉ

  • MessLetters: symbols, emojis, emoticons, and alter your text here

  • ytmp3: converts youtube videos into mp3s

  • EyeDropper: allows you to color pick from anything on your screen

  • placekitten: place holder images but cats :3

  • ezgif: WEBP to GIF, JPG, PNG, etc (I HATE WEBP FILES BURN THEM ALL)

  • Coolors: a color palette generator

  • Image Resizer: crop, resize, and compress images

  • Web page color composer: create a color scheme for your webpage and generate HTML and CSS to be inserted into your page

HTML & CSS Guides

  • w2schools: a great place to start if you're new to coding! teaches HTML, CSS, JavaScript, & more
  • Absolute beginner's guide to Neocities:
    a specific tutorial on how to get started on neocities
  • Learn HTML & CSS:
    a collection of HTML & CSS guides, pretty easy to understand
  • 32-Bit Cafe: an intro to making your own website along with some additional resources
  • Web Design in 4 minutes:
    a hands on guide to web design, also gives you more resources on HTML and CSS
  • HTML Semantic Elements: semantic elements are elements with meaning like "aside" or "article"
  • CSS Terms and Definitions: helpful reference article

Graphics

  • pixel-soup: lots of cute gifs, pngs, backgrounds, and dividers
  • tora*7kg: a variety of graphics made by the artist! (Japanese website)
  • Betty's Graphics: HUGE archive of graphics
  • pixel safari: dividers, favicons, blinkies, buttons, etc.
  • GifCities: archive of old web GIFs
  • Bonnibel's Graphic Collection: I used this a lot to find GIFs n such
  • Glitter-Graphics: more GIFs, dividers, and icons
  • TopeconHeroes: icons, pngs, illustrations (Japanese website; use the buttons at the top to navigate)
  • animated glitter graphics: found a lot of my dividers and lil icons here
  • Tiny Backgrounds: a collection of repeating / tiled backgrounds
  • King-Lulu-Deer: artist that makes pixel art (please credit them if you use their work!)
  • sadgrl.online: more tiled backgrounds I've used (this site also has a lot of HTML resources and more)

Misc. Tips & Help

  • Getting started with Visual Studio Code:
    not a HTML guide BUT it's a good code editor and it's free. highly recommend using VSC
  • File Naming Tips: guide on how you can consistanly name your files
    (do this early on so you don't have to rename all your files later)
  • 32-Bit Cafe Forum: a great community for web hobbyists and professionals
    (has a lot of resources, you can also ask questions and get help on things here)
  • Common HTML Errors & How to Fix / Avoid Them: the name says all, a very helpful resource

Helpful Tools
